ximea_tools
===========

Linux toolbox for XIMEA industrial cameras: a high-throughput recorder, a
PyQt5 GUI, and an integration with the lab's external CueWire hardware
trigger.

.. toctree::
   :maxdepth: 2
   :caption: Contents

   api


Installation
------------

.. code-block:: bash

   pip install -e ".[dev]"

The XIMEA Linux SDK must be installed separately from
https://www.ximea.com/support/wiki/apis/Python — it provides the native
``libm3api`` libraries and the ``ximea`` Python module.
